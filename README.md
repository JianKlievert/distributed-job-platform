# 🚀 Distributed Job Platform

A production-style backend project built to simulate a real-world distributed system while learning modern Backend Engineering and DevOps practices.

## 🎯 Goal

Build, deploy, and operate a production-grade backend system using modern technologies and industry best practices.

### Tech Stack

- **Language:** Python
- **Framework:** FastAPI
- **Database:** PostgreSQL
- **Cache & Queue:** Redis
- **Containerization:** Docker & Docker Compose
- **Cloud:** AWS
- **CI/CD:** GitHub Actions
- **Infrastructure as Code:** Terraform
- **Monitoring:** Prometheus & Grafana
- **Orchestration:** Kubernetes
